# SignLink-AI
A real-time gesture-to-text engine built with Python, OpenCV, and MediaPipe to enhance accessibility. Leveraging a model trained on over 10,000 samples, it achieves 99.90% accuracy in recognizing hand gestures, with future plans to integrate audio output.

Tech Stack
Language: Python
Computer Vision: OpenCV, MediaPipe
Machine Learning: Scikit-learn, Pandas, NumPy
Data Handling: CSV, Pickle
